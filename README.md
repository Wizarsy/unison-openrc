# unison-openrc

## About

Init script for starts unison profile files file with OpenRC

## Usage

You must create the profile file you want to start in `~/.unison`, the interval in seconds for a full scan can be set with `watch_scan_interval` in `${XDG_CONFIG_HOME}/rc/conf.d/unison`.
